Que es

El HTML es el lenguaje de marcado utilizado para crear documentos web. Los estándares web relacionados con HTML incluyen: HTML5: la última versión del lenguaje HTML, que incluye nuevas etiquetas y atributos para mejorar la semántica y la accesibilidad del contenido. XHTML: una versión más estricta de HTML que utiliza una sintaxis XML.
