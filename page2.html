Que es

HTML es el lenguaje utilizado para crear páginas web. Consiste en etiquetas que se utilizan para definir el contenido de la página, como texto, imágenes, listas y vídeos. Inicialmente, el HTML se creó para difundir información textual e imágenes, sin embargo, su uso ha evolucionado y actualmente se utiliza también para fines de entretenimiento y consulta multimedia en línea.

A pesar de que no se previó su uso para fines de entretenimiento y consulta multimedia en línea, se han agregado modificaciones con el tiempo, las cuales se conocen como estándares del HTML. El último estándar, HTML 4.01, fue presentado en febrero de 2001, pero actualmente se utiliza HTML 5, cuyo borrador casi definitivo se presentó en mayo de 2005.

El HTML es un lenguaje de marcación de elementos para crear documentos hipertexto. Aunque es fácil de aprender y cualquier persona puede crear una página web con él, para obtener resultados profesionales se necesitan habilidades de diseño y el uso de otros lenguajes como CSS. El código HTML se escribe en un archivo de texto con extensión .html o .htm utilizando un editor de texto sencillo.

Marcador HTML

HTML utiliza etiquetas encerradas en corchetes angulares para describir la estructura de un documento. Además, HTML también puede especificar la apariencia del documento y puede incluir scripts que afectan el comportamiento de los navegadores web y otros procesadores de HTML.

Primeras especificaciones de HTML

En 1991, Tim Berners-Lee propuso 18 elementos para el diseño inicial de HTML, de los cuales 13 siguen siendo utilizados en la versión HTML 4.4. Aunque Berners-Lee consideraba HTML una extensión de SGML, no fue formalmente reconocido como tal hasta mediados de 1993.

Berners-Lee y Dan Connolly propusieron el primer borrador del HTML, que incluía una Definición de Tipo de Documento SGML para definir la gramática.
