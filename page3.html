Que es CSS?

CSS es la abreviatura de "Cascading Style Sheets" o "hojas de estilo en cascada" en español. Se trata de un lenguaje utilizado para dar estilo a los elementos escritos en HTML.

El lenguaje CSS fue desarrollado por W3C (World Wide Web Consortium) en 1996 con el objetivo de solucionar un problema que HTML no había previsto: la falta de etiquetas para formatear la página. Antes de CSS, los desarrolladores tenían que incluir etiquetas en HTML, como font, lo que generaba problemas a la hora de crear un sitio web con diferentes fuentes, fondos y estilos. La reescritura del código era larga, dolorosa y costosa. Por lo tanto, se creó CSS para ofrecer una solución a esta dificultad.

relación entre HTML y CSS es muy importante, ya que HTML es el lenguaje de marcado que constituye la base de un sitio web y CSS se encarga de toda la parte estética del mismo.

CSS no es técnica y necesariamente obligatoria para el funcionamiento de un sitio web, es fundamental para dar estilo y coherencia al diseño visual del mismo. De hecho, un sitio web que solo tenga HTML y carezca de CSS se verá muy básico y desordenado.
